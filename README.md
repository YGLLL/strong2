**strong 是 Android 平台上一款观看短视频的App 。
项目基于dkplayer，并采用Android官方缓存技术实现。
用户可以在strong内观看短视频，可以通过上下滑动切换视频，并享受视频间的流畅切换。
数据来自bilibili。**

## 项目特点
- 短视频风格的界面，可以向下无限滑动
- 采用Android官方缓存技术，实现视频间的流畅切换
- 使用来自bilibili的真实数据

## 项目使用的开源库
- dkplayer
- retrofit2
- litepal
- glide
